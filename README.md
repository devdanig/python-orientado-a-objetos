## OOP Concepts Covered
This repository focuses on mastering core and advanced OOP concepts in Python:
* **Classes and Objects**
* **Constructors** (`__init__`)
* **Instance methods** and special methods (`__str__`)
* **Encapsulation** with private attributes and properties (`@property`)
* **Inheritance** to reduce code duplication
* **Polymorphism** using interfaces and protocols
* **Composition** over inheritance
* **Abstract Base Classes** (`ABC`, `@abstractmethod`)
* **Static and class methods**
* **Exception handling** (`try`, `except`, `raise`)
* **Object serialization** and persistence using JSON

## Virtual Environments
All examples assume the use of a virtual environment to keep dependencies isolated and portable.

**1. Create a virtual environment:**
`python -m venv venv`

**2. Activate it:**
* **Windows:** `.\venv\Scripts\activate`
* **macOS / Linux:** `source venv/bin/activate`

**3. Deactivate:**
`deactivate`

*Note: The `venv/` folder is ignored via `.gitignore` and should never be committed.*
